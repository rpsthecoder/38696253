How to turn emojis into icons

Granted there are many online resources (both free and paid) for icons, we still should be able to use the one style of symbols we already have and are uber familiar with — emojis — as icons.

Icons are shapes, typically, made of a single color block. This can be achieved with emojis using text-shadow! 

.icon {
  color: transparent;
  text-shadow: 0 0 black;
}

Check out a Codepen demo here

The transparent color hides the actual emoji and in its place we see the shadow behind.

You can also use background-clip instead of the above method. 

.icon {
  color: transparent;
  background-color: black;
  background-clip: text;
}

With background-clip, if you use gradient backgrounds for the icons, you'll get gradient icons.
